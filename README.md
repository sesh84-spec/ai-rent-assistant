# AI-ассистент для автоматизации аренды домов (VolgaDom Rent)

## Описание проекта

AI-ассистент для автоматизации обработки заявок на аренду загородных домов.  
Система анализирует сообщения клиентов, подбирает варианты домов и автоматически создаёт или обновляет сделки в CRM.

---

## Цель проекта

- Автоматизировать обработку входящих заявок  
- Снизить нагрузку на менеджеров  
- Ускорить ответы клиентам  
- Стандартизировать сбор данных  

---

## Архитектура
Клиент → Wazzup → Albato → OpenAI → JSON → Router → amoCRM → Ответ клиенту


---

## Как работает система

1. Клиент пишет сообщение в Telegram  
2. Сообщение поступает в Albato через Wazzup  
3. Проверяется контакт и сделка в amoCRM  
4. OpenAI анализирует запрос  
5. Формируется JSON с параметрами заявки  
6. Router определяет сценарий обработки  
7. В зависимости от сценария:
   - создаётся новая сделка  
   - или обновляется существующая  
8. Создаётся задача менеджеру  
9. Клиент получает ответ  

---

## Реализованные сценарии

- Новая заявка (NEWLEAD)  
- Изменение заявки  
- Изменение брони  
- Отмена брони  
- Новая бронь при наличии  

Особенность:  
при сценарии «новая бронь при наличии» создаётся отдельная сделка, а не перезаписывается существующая.
